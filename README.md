# Sky Runner 3D 🏃☁️

เกมวิ่งเก็บเหรียญ 3D สไตล์ Cookie Run สร้างด้วย [Three.js](https://threejs.org/) ล้วนๆ — ไม่ต้องติดตั้งอะไร เปิด `index.html` ในเบราว์เซอร์แล้วเล่นได้เลย (ออฟไลน์ก็ได้)

## วิธีเล่น

| ปุ่ม | การกระทำ |
|---|---|
| `←` `→` หรือ `A` `D` | เปลี่ยนเลน |
| `↑` / `W` / `Space` | กระโดด (กดซ้ำกลางอากาศ = ตีลังกา 2 ชั้น) |
| `↓` / `S` | สไลด์ลอดคาน / ดิ่งลงเร็วกลางอากาศ |
| ปัดจอ (มือถือ) | ซ้าย/ขวา/ขึ้น/ลง |

## ระบบในเกม

- **เลือดแบบ Cookie Run** — HP ลดตลอดเวลา ชนสิ่งกีดขวางเสียเลือดแต่วิ่งต่อได้ เก็บ ❤️ เพื่อฟื้น
- **ไอเทม** — 🧲 แม่เหล็กดูดเหรียญ, 🛡️ โล่กันชน, 🚀 บูสต์พุ่งทะลุ, 🧪 ร่างยักษ์
- **Bonus Time** — วิ่งครบระยะเข้าโหมดฝนเหรียญ ท้องฟ้าสีทอง เหรียญค่า x2
- **เงินสะสมถาวร + ร้านค้า** — อัพเกรด HP / ตัวคูณแต้ม / ความยาวโบนัสไทม์
- **ตัวละคร 5 ตัว** — แต่ละตัวมีความสามารถเฉพาะ (นินจา, อัศวิน, แม่มด, จอมโจร)
- **เพื่อนซี้ (Pet) 4 ตัว** — ลอยตามตัวช่วยออกสกิลอัตโนมัติ
- **ฉาก 4 ธีม** — ทุ่งเมฆ → ป่ามรกต → ภูเขาไฟ → อวกาศ เปลี่ยนตามระยะทาง
- **เพลง chiptune + เสียงเอฟเฟกต์** — สังเคราะห์สดด้วย WebAudio ไม่ใช้ไฟล์เสียง

## โครงสร้างไฟล์

```
index.html    หน้าเกม + HUD + ร้านค้า
game.js       โค้ดเกมทั้งหมด
three.min.js  ไลบรารี Three.js (r147)
```

เซฟเกม (เงิน, อัพเกรด, ตัวละคร, สถิติ) เก็บใน `localStorage` ของเบราว์เซอร์
